# 🤖 AI Resume Analyzer
Upload a resume PDF and get role-specific ATS feedback using AI.

## Features:
- Resume upload
- Role-based analysis
- Job description matching
- Missing skills detection
- ATS-style critique

## Tech Stack:
- Python
- Streamlit
- OpenRouter
- DeepSeek
- PyPDF2
